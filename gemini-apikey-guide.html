🤔 什麼是 API Key？

API Key 就像一把「鑰匙」，讓我們的陪跑系統能去敲 Google Gemini AI 的門，請它幫你分析學習狀況。每個人要有自己的鑰匙，資料才不會混在一起。這把鑰匙只存在你自己的手機／電腦裡，不會被別人看到。

申請步驟
跟著做，每一步都有畫面說明
1

用 Google 帳號登入 AI Studio

打開瀏覽器，前往 Google AI Studio（用你的 Gmail 帳號登入就好，學校 Google 帳號或私人 Gmail 都可以）：

🔗 aistudio.google.com
Get Started
↑ 第一次進入點這個藍色按鈕，選擇你的 Google 帳號
💡 提示：第一次使用會跳出「同意服務條款」，勾選同意後按 Continue 即可。系統會問你所在國家，選 Taiwan。
2

點左側選單的「Get API Key」

登入後，看畫面左邊的選單，往下找到一個鑰匙圖示的選項：

🔑 Get API Key
↑ 通常在左側選單偏下方的位置

點下去就會進到「API Keys」管理頁面。

3

點右上角「Create API Key」建立鑰匙

在 API Keys 頁面的右上角，點藍色按鈕：

+ Create API Key
↑ 右上角藍色按鈕

它會請你選一個「專案（Project）」。如果你是第一次用，直接選 Create API key in a new project（在新專案建立）最簡單。

⚠️ 如果出現 "Project creation is disabled" 錯誤：這是 Google 偶爾的限制。解法：先到 console.cloud.google.com 隨便建一個新專案（取名如 my-gemini，不用綁信用卡），再回到這裡選那個專案即可。
4

複製你的鑰匙！

建立成功後，畫面會出現一串以 AIzaSy 開頭的長字串，這就是你的 API Key。點旁邊的複製圖示 📋把它複製起來：

AIzaSyXXXXXXXX–你的專屬鑰匙–XXXXXXXX
↑ 長這樣，按右邊複製鈕
🔒 重要：這把鑰匙等於你的帳號授權，不要截圖貼到網路上、不要傳給別人。貼到自己的備忘錄存好就行。
5

貼進學測陪跑系統，完成！

回到你的「學測陪跑系統」網頁，點下方的 🤖 AI教練 分頁，把剛剛複製的鑰匙貼進「Gemini API Key」欄位，按 啟動 AI 教練診斷，就能得到你的專屬學習報告了 🎉

✓ 一次設定，長期使用：鑰匙會記在你的瀏覽器裡，之後同一台裝置、同一個瀏覽器打開就不用再貼。
常見問題 FAQ
Q真的免費嗎？會不會被扣錢？▾
免費版完全不用綁信用卡，也不會扣錢。Google 提供的免費額度（每分鐘約 10 次、每天約 250 次請求）對於每天做一次學習覆盤來說綽綽有餘。除非你主動去「設定付款」綁卡升級，否則不可能產生費用。
Q申請需要多久？要審核嗎？▾
不用審核，整個流程 3 分鐘內完成，鑰匙是「即時」產生的，複製貼上馬上就能用。
Q一個帳號可以申請幾把鑰匙？▾
同一個 Google 帳號可以建立多把（最多 10 個專案）。但對學測陪跑系統來說，一把就夠用了。
Q貼上鑰匙後出現錯誤怎麼辦？▾
先確認三件事：① 鑰匙完整複製（開頭是 AIzaSy，沒有多空格）② 你的網路正常 ③ 試著在系統裡切換成 gemini-2.5-flash-lite 模型再試一次。若還是不行，可能是當天免費額度用完，隔天會重置。
Q換手機 / 換電腦鑰匙會不見嗎？▾
鑰匙存在你「當下使用的瀏覽器」裡，換裝置或清除瀏覽紀錄後需要重貼一次。建議把鑰匙存在自己的備忘錄裡備用。
